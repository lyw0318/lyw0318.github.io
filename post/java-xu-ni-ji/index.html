深入理解JAVA虚拟机

2020-02-04

1. JDK构成


JVM(Java Virtual Machine)：
java虚拟机，java程序的核心执行器。
JRE(Java Runtime Environment)：
java运行时环境，JVM+java的核心类库
JDK(Java Development Kit)：
java开发工具包，JRE+java开发工具

2. JVM构成


1.类加载器（Class Loader）：加载类文件到内存。
2.运行时数据区（Runtime Data Area）：虚拟机内存分部。
3.执行引擎（Execution Engine）：也叫解释器，负责解释命令，交由操作系统执行。
4.本地接口（Native Interface）： 调用其他语言库的或方法的出口。

2.1. JVM内存管理

内存管理是JVM对运行时数据区进行管理的子系统。因为可用内存是有限的，所以回收复用内存十分必要。
